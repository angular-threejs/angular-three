Angular Three Cannon is an integration of the [Cannon.js](https://github.com/pmndrs/cannon-es) physics engine for use with Angular Three.

This implementation is based on the [@react-three/cannon](https://github.com/pmndrs/use-cannon) library.

- [x] Doesn't block the main thread, runs in a web worker
- [x] Supports all the features of cannon-es

Examples are available at [angular-three-cannon](https://demo.angularthree.org/cannon)

## Installation

```sh
npm install angular-three-cannon cannon-es @pmndrs/cannon-worker-api
# yarn add angular-three-cannon cannon-es @pmndrs/cannon-worker-api
# pnpm add angular-three-cannon cannon-es @pmndrs/cannon-worker-api
```
